# Documentation

This directory contains all project documentation for the Virtual Table Top (VTT) project.

## Structure

- `reports/` - Implementation reports and completion summaries
- `guides/` - User guides, developer guides, and reference materials
- `session_notes/` - Development session notes and progress tracking
- `architecture/` - System architecture and design documentation

## Quick Links

- [System Overview](architecture/SYSTEM_OVERVIEW.md)
- [Database Design](architecture/DATABASE_DESIGN.md)

## Documentation Standards

- Use Markdown format for all documentation
- Keep documentation up-to-date with code changes
- Include diagrams where appropriate
- Follow the naming convention: `YYYY-MM-DD-description.md` for dated documents
