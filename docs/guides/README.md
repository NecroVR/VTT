# Guides and References

This directory contains user guides, developer guides, and reference documentation for the VTT project.

## Purpose

Guides provide step-by-step instructions and reference materials for:
- Setting up the development environment
- Using the application features
- Contributing to the project
- API documentation
- Troubleshooting common issues

## Types of Guides

- **User Guides** - End-user documentation for application features
- **Developer Guides** - Setup, architecture, and contribution guidelines
- **API References** - Endpoint documentation and usage examples
- **Troubleshooting Guides** - Common issues and solutions
- **Best Practices** - Coding standards and design patterns

## Naming Convention

Use clear, descriptive names:
- `GETTING_STARTED.md`
- `API_REFERENCE.md`
- `TROUBLESHOOTING.md`
- `CONTRIBUTING.md`
